# Variables

With javascript we can declare a new variable with the keyword `var`.

```js
var firstName = "Bob";
var age = 42;
var isCoder = true
var decimal = 3.14
var birthday = new Date(1990, 1, 1)
var person = {}
var empty = undefined
```

We can then print that variable using `console.log`

```js
var firstName = "Bob";
var age = 42;
var isCoder = true
var decimal = 3.14
var birthday = new Date(1990, 1, 1)
var person = {}
var empty = undefined

console.log(firstName)
console.log(age)
console.log(isCoder)
console.log(decimal)
console.log(birthday)
console.log(person)
console.log(empty)
```

Try creating and printing some variables.
